# Create Telegram Bot

## Step 1: Open BotFather

1. Open Telegram
2. Search for `@BotFather`
3. Start a chat

## Step 2: Create Bot

Send these messages:

```
/newbot
```

BotFather: "What name?"
```
The Phantom
```

BotFather: "Choose username"
```
ThePhantomBot
```
(Must end in `bot`)

## Step 3: Save Token

BotFather responds with:
```

123456789:ABCdefGHIjklMNOpqrsTUVwxyz
```

**SAVE THIS TOKEN! Keep it secret!**
